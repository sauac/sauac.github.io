The team

Welcome to the team of  One.

Currently I'm the only member, the sole dreamer.

I started this website on a Saturday, while at work as a receptionist at a dental surgery.
It was a quite day and I'd recently started learning HTML, so I decided to practice what I'd learnt so far.

"Oh I know!", I thought.

I'll create a card; the kind with an image on top and text below with rounded corners and a shadow underneath. I quickly searched the web, found a great image and after some CSS styling I'd created a card; which I named "peaceful Autumn".

So began the "four seasons".

I'm still learning web design and have decided to use this site as a platform for my learning and creativity. I'll write whatever comes out of my heart and mind, framing the output into sections and pages.

Hopefully, over time, I'll have written down a collection of thoughts, feelings and idea's, that I or anyone else, can enjoy reading!

...ok, so this is what blogging is!
