The team

Welcome to the team.

It's a team of one, me, as currently I'm the only member.

I started this website on a Saturday, while working as a receptionist at a dental surgery.
It was a quiet day and the reception was unusualy empty. I'd recently started learning HTML at home, so I decided to practice what I'd learnt so far. I opened notepad, typed in the standard HTML layout "head, body etc and saved the file as work_doodle.html.

After a short while I decided "I'll create a card"; the kind with an image on top and text below, with rounded corners and a shadow underneath. I quickly searched the web, found an image I liked then wrote some text that described the image. After adding some CSS styling, the basic design of the card was complete!

In the center of the card I added a heading, "peaceful Autumn" and so began the "four seasons" website.

I'm still learning web design and have decided to use this site as a platform for my learning and creativity. I'll write whatever comes out of my heart and mind, framing the output into sections and pages.

Over time, as well as learning to code, I'll have written down a collection of thoughts, feelings and idea's, that I and hopefully others can enjoy reading!

...ok, so this is what blogging is!
